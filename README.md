## 🔬 About ARC

The **Automated Robotics for Chemistry (ARC)** lab, led by [Professor Ian Fairlamb](https://www.york.ac.uk/chemistry/people/ifairlamb/), develops and deploys robotics and automation systems to accelerate chemical research at the University of York.

We combine **bespoke in-house software and hardware** with commercial platforms to enable:

- 🧪 Solid and liquid handling automation
- ⚡ High-throughput reactions (48 and 96-well formats)
- 💡 Photo-catalysis and flow chemistry
- 📊 Automated LC-MS and NMR dataset processing
- 📈 Reaction outcome trend analysis via data science tools

| **Radley's Carousels** | Multi-position reaction carousel systems |
| **Dobot Robotic Arms** | Programmable robotic manipulation and sample handling |

## 💻 Our Software & Automation Projects

This GitHub organisation hosts open-source tools and control systems developed in-house at ARC. Our projects span:

- **Instrument Control** — Python-based interfaces for laboratory devices (HPLC autosamplers, hotplates, syringe pumps)
- **Robotic Arm Integration** — TCP/IP communication systems for Dobot robotic arms, including gripper and syringe pump control
- **Data Pipelines** — Automated processing of LC-MS and NMR datasets
- **Lab Automation GUIs** — Tkinter and web-based interfaces for equipment control and monitoring
